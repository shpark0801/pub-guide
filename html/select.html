select

01. select - 기본 

<div class="form-row">
    <label for="selectBasic" class="form-row-label">기본형</label>
    <div class="form-row-control">
        <select id="selectBasic" class="form-select">
            <option value="">선택하세요</option>
            <option value="option1">옵션 1</option>
            <option value="option2">옵션 2</option>
        </select>
    </div>
</div>
기본형은 브라우저 기본 UI를 사용하며, 화살표의 위치와 형태는 브라우저 및 운영체제에 따라 다르게 표시될 수 있다.

02. select - 화살표형 

<div class="form-row">
    <label for="selectArrow" class="form-row-label">화살표형</label>
    <div class="form-row-control">
        <div class="form-select-wrap">
            <select id="selectArrow" class="form-select form-select-arrow">
                <option value="">선택하세요</option>
                <option value="option1">옵션 1</option>
                <option value="option2">옵션 2</option>
            </select>
        </div>
    </div>
</div>
커스텀 화살표를 사용하는 select는 .form-select-wrap으로 감싸고, select에 .form-select-arrow를 적용한다.

05. select - 세로형 

<div class="form-field">
    <label for="selectVertical" class="form-label">선택</label>
    <div class="form-select-wrap">
        <select id="selectVertical" class="form-select form-select-box">
            <option value="">선택해주세요</option>
            <option value="option1">옵션1</option>
            <option value="option2">옵션2</option>
            <option value="option3">옵션3</option>
        </select>
    </div>
</div>
세로형은 .form-field 안에 .form-label과 select를 배치한다.
좌우형은 .form-row 구조를 사용한다.

06. select - 필수 선택 

<div class="form-field">
    <label for="selectRequired" class="form-label">
        문의 유형 <span class="form-required" aria-hidden="true">*</span>
    </label>
    <div class="form-select-wrap">
        <select
            id="selectRequired"
            class="form-select form-select-box"
            required
            aria-required="true">
            <option value="" selected disabled>선택해주세요</option>
            <option value="service">서비스 문의</option>
            <option value="payment">결제 문의</option>
            <option value="etc">기타 문의</option>
        </select>
    </div>
</div>
필수 선택 항목에는 required를 사용한다.
안내용 첫 option은 value를 비우고 selected, disabled를 적용해 실제 값으로 제출되지 않도록 구성한다.

07. select - 옵션 그룹 

<div class="form-field">
    <label for="selectOptgroup" class="form-label">지역 선택</label>
    <div class="form-select-wrap">
        <select id="selectOptgroup" class="form-select form-select-box">
            <option value="">지역을 선택해주세요</option>
            <optgroup label="수도권">
                <option value="seoul">서울</option>
                <option value="gyeonggi">경기</option>
                <option value="incheon">인천</option>
            </optgroup>
            <optgroup label="충청권">
                <option value="daejeon">대전</option>
                <option value="sejong">세종</option>
                <option value="chungcheong">충청</option>
            </optgroup>
        </select>
    </div>
</div>
관련 옵션이 많을 경우 optgroup의 label로 항목을 구분한다.
optgroup 자체는 선택할 수 없으며 하위 option만 선택된다.

08. select - 상태별

select - 에러

문의 유형을 선택해주세요. 

<div class="form-field is-error">
    <label for="selectError" class="form-label">문의 유형</label>
    <div class="form-select-wrap">
        <select
            id="selectError"
            class="form-select form-select-box"
            aria-invalid="true"
            aria-describedby="selectErrorMessage">
            <option value="">선택해주세요</option>
            <option value="service">서비스 문의</option>
            <option value="payment">결제 문의</option>
        </select>
    </div>
    <p id="selectErrorMessage" class="form-msg form-msg-error">
        문의 유형을 선택해주세요.
    </p>
</div>

select - 경고

선택한 지역은 추가 배송비가 발생할 수 있습니다. 

<div class="form-field is-warn">
    <label for="selectWarning" class="form-label">배송 지역</label>
    <div class="form-select-wrap">
        <select
            id="selectWarning"
            class="form-select form-select-box"
            aria-describedby="selectWarningMessage">
            <option value="">선택해주세요</option>
            <option value="seoul">서울</option>
            <option value="jeju" selected>제주</option>
        </select>
    </div>
    <p id="selectWarningMessage" class="form-msg form-msg-warn">
        선택한 지역은 추가 배송비가 발생할 수 있습니다.
    </p>
</div>

select - 비활성

<div class="form-field">
    <label for="selectDisabled" class="form-label">선택 불가</label>
    <div class="form-select-wrap is-disabled">
        <select
            id="selectDisabled"
            class="form-select form-select-box"
            disabled>
            <option value="">선택할 수 없습니다</option>
        </select>
    </div>
</div>
상태 유형은 부모 .form-field에 .is-error 또는 .is-warn을 적용한다.
비활성 상태는 select에 disabled를 사용한다.
select 요소는 readonly 속성을 지원하지 않는다.
