progress

01. progress - 기본

업로드 진행률 60% 
<!-- progress 기본 예제 -->
<div class="ui-progress">
    <div class="ui-progress-info"><span>업로드 진행률</span><span>60%</span></div>
    <div class="ui-progress-track" role="progressbar" aria-label="파일 업로드 진행률" aria-valuemin="0" aria-valuemax="100" aria-valuenow="60">
        <span class="ui-progress-bar" style="width: 60%;"></span>
    </div>
</div>
  • 최초 노출 시 왼쪽에서 목표 진행률까지 채워지고, 이후 width 변경은 부드럽게 전환된다.
  • 진행률이 변경되면 bar의 width, 표시 문구, aria-valuenow 값을 동일하게 갱신한다.
  • 모션 축소 설정에서는 애니메이션 없이 최종 진행률을 바로 표시한다.

02. progress - fluid

전체 작업 40% 
<div class="ui-progress ui-progress-fluid">
    <div class="ui-progress-head"><span>전체 작업</span><strong>40%</strong></div>
    <div class="ui-progress-track" role="progressbar" aria-label="전체 작업 진행률" aria-valuemin="0" aria-valuemax="100" aria-valuenow="40">
        <span class="ui-progress-bar" style="width: 40%;"></span>
    </div>
</div>
기본 최대 너비를 유지하고, 폼이나 콘텐츠 폭을 모두 사용해야 할 때만 .ui-progress-fluid를 추가한다.

01. loading - 기본

불러오는 중 
<!-- loading 기본 예제 -->
<div class="ui-loading" role="status">
    <span class="ui-loading-spinner" aria-hidden="true"></span>
    <span>불러오는 중</span>
</div>
종료 시 로딩 요소를 DOM에서 제거하거나 숨긴다. 상태 문구는 시각적으로도 제공한다.
