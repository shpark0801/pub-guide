radio

01. radio - 기본 

<div class="form-radio">
    <input type="radio" id="radio-default" name="radio-default" class="form-radio-input" />
    <label for="radio-default" class="form-radio-label">기본 라디오</label>
</div>

02. radio - 커스텀 

<div class="form-radio form-radio-custom">
    <input type="radio" id="radio-custom" name="radio-custom" class="form-radio-input" />
    <label for="radio-custom" class="form-radio-label">커스텀 라디오</label>
</div>
커스텀 라디오는 기본 구조에 .form-radio-custom 클래스를 추가한다.

04. radio - 그룹

배송 방법 
<fieldset class="form-fieldset">
    <legend class="form-legend">배송 방법</legend>

    <div class="form-option-group">
        <div class="form-radio form-radio-custom">
            <input
                type="radio"
                id="radio-group-01"
                name="delivery-method"
                class="form-radio-input"
                checked
            />
            <label for="radio-group-01" class="form-radio-label">일반 배송</label>
        </div>

        <div class="form-radio form-radio-custom">
            <input type="radio" id="radio-group-02" name="delivery-method" class="form-radio-input" />
            <label for="radio-group-02" class="form-radio-label">빠른 배송</label>
        </div>

        <div class="form-radio form-radio-custom">
            <input
                type="radio"
                id="radio-group-03"
                name="delivery-method"
                class="form-radio-input"
                disabled
            />
            <label for="radio-group-03" class="form-radio-label">방문 수령</label>
        </div>
    </div>
</fieldset>
동일한 질문에 속하는 라디오 버튼은 같은 name 값을 사용하고 fieldset과 legend로 묶는다.
라디오 버튼은 동일한 그룹에서 하나의 항목만 선택할 때 사용한다.
