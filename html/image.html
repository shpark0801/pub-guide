09. image 참고

  • 목록·카드처럼 동일한 비율이 필요한 이미지는 cover, 로고·상품처럼 전체 형태가 중요하면 contain을 사용한다.
  • width와 height 속성으로 이미지 영역을 확보하고 첫 화면 밖의 이미지에는 loading="lazy" 적용을 검토한다.
  • 정보성 이미지는 의미 있는 alt를 작성하고 장식 이미지는 alt=""로 제공한다.
  • 화면별 이미지가 다르면 picture와 source를 사용하며 정보성 이미지를 CSS 배경으로만 제공하지 않는다.
